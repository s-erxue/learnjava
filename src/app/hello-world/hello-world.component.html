Hello World

The first program we will write is a program that says hello, world.

Creating a new file

Right click the java folder and select New > Java class. Name it Hello.

public class Hello {
}

So far, this code doesn't do anything. First, let's break it apart. In Java, there is something called a class. Objects, like integers and strings (text) are implemented with classes. You can create an instance of a class. You can think of the class as a blueprint and the instance as a house. Instances can have methods, which can do things with them. Classes can also have static methods, which mean that they don't do something with an instance, but they can still do things. Some classes are not supposed to be blueprints. They are there to provide helper static methods. Our program, Hello, has a class with the same name. We need to run the program. For that, we need some more code. To define a class, we can use [visibility] class [name]. The visibility is when you are writing code for others to use. The default, package private, is only accessible within your code. Public means it's accessible everywhere. We declare all the methods inside curly braces. It is proper style to indent everything inside braces. The IDE will add some space before your method, and you shouldn't delete it. Here is the method. 

public class Hello {
    public static void main(String[] args) {
    }
}

Methods are more complicated. To define the part of your program that is executed, you can use public static void main(String[] args). The code inside main is also inside braces, with more indentation. We can click the green play button on the first line, and it will run the program. You can try to run it yourself, however, it will do nothing. To print something to the console, we need more code. 

public class Hello {
    public static void main(String[] args) {
        System.out.println("hello, world");
    }
}

When you run this, you will see hello, world in the console.
